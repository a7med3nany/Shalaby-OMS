# 📦 Shalaby OMS - نظام إدارة الطلبات الذكي

![Version](https://img.shields.io/badge/version-1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

نظام احترافي لإدارة الطلبات مصمم خصيصاً للأعمال التجارية المصرية.

---

## ✨ المميزات

- 🎨 **تصميم عصري**: واجهة مستخدم حديثة بتأثيرات Glassmorphism
- 📱 **متجاوب تماماً**: يعمل بكفاءة على جميع الأجهزة (موبايل، تابلت، ديسكتوب)
- ⚡ **سريع وخفيف**: أداء ممتاز بدون مكتبات ثقيلة
- 🔒 **آمن**: بنية نظيفة ومنظمة
- 🌐 **باللغة العربية**: واجهة كاملة باللغة العربية
- 📊 **تكامل مع Google Sheets**: حفظ البيانات بشكل آمن

---

## 🚀 البدء السريع

### المتطلبات

- حساب Google (للتكامل مع Sheets)
